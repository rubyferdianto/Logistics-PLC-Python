🔧 Step 1: PLC → SCADA (OPC UA)

OPC UA Node Data:
• Structured object nodes
• Data type validation
• Quality & timestamp info
• Secure encrypted transport
• Standardized addressing

📊 Step 2: SCADA → MES (REST API)

REST API Communication:
• HTTPS JSON payloads
• Bearer token authentication
• Real-time data posting
• Response validation
• Error handling & retries

⚙️ Step 3: MES → Digital Twin (SQL Database)

SQL Database Integration:
• Shared enterprise database access
• ODBC/JDBC connections
• Real-time database triggers
• SQL queries and views
• Enterprise data warehouse

🌐 Step 4: Digital Twin → PLC (OPC UA Write)

OPC UA Command Writing:
• OPC UA Client write operations
• Structured command data
• Real-time robot instructions
• Secure bidirectional protocol
• Industrial-grade reliability

🔄 Step 5: PLC Response

Robot Command Data:
• Target zone coordinates
• Navigation waypoints
• Action commands
• Movement instructions
• Execution confirmations
